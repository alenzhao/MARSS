MARSS User Guide

The MARSS user guide presents the theory behind MARSS models and estimation of model parameters. It discusses how models are specified in the package and what model variants can be fit. It has case studies that walk you though different types of ecological analyses using the MARSS package.

Example Scripts

The user guide has case studies which will walk you through a number of analyses. The code for the case studies can be opened in R by typing RShowDoc("Case_study_1.R",package="MARSS") at the R command. Replace "1" with 2, 3, 5 to see the other case study codes.
